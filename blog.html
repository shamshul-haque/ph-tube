Discuss the scope of var, let and const

  • var: It is a functional scoped. It can be updated and re-declared.
  • let: It is a block scoped. It can be updated but can not be re-declared.
  • const: It is also a block scoped but it can not be updated and re-declared.

Tell us the use cases of null and undefined

  • null: It is a value that represents the intentional absence of a value. It is assigned explicitly. It's type is object.
  • undefined: It occurs when a variable has been declared but the value is not assigned to the variable. It's type is undefined.

What do you mean by REST API?

A complete form of REST API is Representational State Transfer Application Programming Interface which is an architectural approach for creating web services using standard HTTP methods and resource-based URLs. It's widely used in web and mobile app development for data communication.
